HTML

The Hyper Text Markup Languageor HTML is a standard markup language for documents design to be displayed in a browser.It is defines meaning and structure of web content

CSS

Cascading Style Sheets(css) is used to style and design web pages. It controls the layout, colors, fonts, and spacing of HTML elements. With CSS, we can make web pages look attractive and responsive on all devices.

JavaScript

JavaScript, is a programming language used to make web pages interactive and dynamic. It allows you to add features like animations, form validations, pop-ups, and interactive buttons. JavaScript works with HTML and CSS to create engaging user experiences.
